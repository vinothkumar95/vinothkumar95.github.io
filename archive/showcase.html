News Content Classifier

Python, Mongo, Celery, Scikit Learn

A django-python based News content classifier which runs a cron using celery to scrape news content from various sites using beautiful soup and classifies them with scikit learn based on their trained data. The scraped data was trained manually, as no Indian news content dataset is available. Time consuming scraping operations were made to run in the background as async tasks with redis server.
